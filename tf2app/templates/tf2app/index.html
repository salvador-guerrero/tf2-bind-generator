TF2 Text to Bind Generator

How to use
  • Copy your text into the big box on top
  • Push the Generate button
  • Copy the generated script into a new file called autoexec.cfg
  • Save that file at this location (Create new folders if missing) C:\Program Files\Steam\steamapps\common\Team Fortress 2\tf\custom\texttobind\cfg
  • Done! Open TF2 and have fun
